# 🚀 STEP 3: 무료 호스팅 배포(Deploy) 가이드라인

작성된 코드를 실제 인터넷 상(나만의 도메인)에 올리기 위한 가장 중요하고 마지막 단계입니다. 코드를 모두 서버에 올린 뒤 스마트폰으로 접속해보세요!

---

## 🚩 선수 작업: GitHub에 코드 올리기 (필수)
Vercel과 Render 모두 사용자님의 GitHub를 통해 원클릭 배포가 됩니다.
1. `c:\python\tennis-web-project` 폴더 안의 내용물을 통째로 본인의 GitHub 새 Repository에 업로드(Commit & Push) 해줍니다.
2. (이때 `frontend/node_modules` 같은 무거운 폴더나 `.env` 같은 비밀 정보는 올라가지 않도록, 나중에 서버에서 `.env` 환경 변수만 따로 입력합니다.)

---

## 🛠️ 1. 백엔드 (Render.com) 배포하기
이곳에 봇(`bot.py`)을 돌릴 서버를 구성합니다.

1. **로그인**: [Render.com](https://render.com)에 접속하여 GitHub 아이디로 로그인합니다.
2. 우측 상단의 **New +** 버튼을 누르고 **Web Service**를 선택합니다.
3. 방금 GitHub에 올린 `tennis-web-project` 저장소를 선택합니다.
4. 아래와 같이 설정합니다:
   - **Name**: `tennis-reserve-server` (아무 이름이나)
   - **Root Directory**: `backend` (중요!)
   - **Environment**: `Python 3`
   - **Build Command**: `./build.sh` (아까 만든 크롬 설치 및 패키지 설치 스크립트)
   - **Start Command**: `uvicorn server:app --host 0.0.0.0 --port $PORT`
   - **Instance Type**: `Free`
5. **Environment Variables (환경 변수) 설정**:
   - `Advanced` 탭의 `Add Environment Variable`을 눌러 기존의 `.env` 내용을 입력합니다.
     - `TELEGRAM_TOKEN` : (봇 토큰)
     - `TELEGRAM_USER_ID` : (유저 ID)
     - `PYTHON_VERSION` : `3.10.0` (가장 호환성 좋음)
     - `PYTHONUNBUFFERED` : `1` (로그를 실시간으로 보기 위해 필수!)
6. 가장 아래의 **Create Web Service** 버튼을 누르면 배포가 시작되며 약 3~5분 후 크롬까지 완벽하게 설치된 무료 서버가 탄생합니다!
   - 배포가 완료되면, `https://tennis-web-project.onrender.com` 와 같은 나만의 백엔드 주소가 생깁니다. (이 주소를 꼭 복사해 두세요!)

---

## 🌐 2. 프론트엔드 (Vercel) 배포하기
이제 내가 누를 화면인 리모컨(Next.js 웹)을 찍어낼 차례입니다.

1. 앞서 복사한 Render 백엔드 주소를 바탕으로 코드를 딱 한 줄 수정해야 합니다. 
   - `frontend/src/app/page.tsx` (없을 경우 `frontend/app/page.tsx`) 파일을 에디터로 열어서 10번째 줄 부근의 `API_URL`을 아까 배포한 렌더 주소로 교체합니다.
   - 수정 전: `const API_URL = 'http://localhost:8000';`
   - 수정 후: `const API_URL = 'https://tennis-web-project.onrender.com';` (본인의 실제 배포 주소로 변경하세요)
   - 수정한 내용을 다시 GitHub에 Push 합니다.

2. **로그인**: [Vercel.com](https://vercel.com)에 접속해서 GitHub로 로그인합니다.
3. **Add New Project**를 누르고 동일하게 `tennis-web-project` 저장소에서 **Import** 버튼을 클릭합니다.
4. 설정 화면 중간에 나오는 **Root Directory** 옆의 `Edit`를 눌러 `frontend` 폴더를 선택합니다.
5. 그대로 **Deploy** 버튼을 누르면 끝입니다!
6. 약 1분 후 배포가 완료되면 `https://frontend-어쩌구저쩌구.vercel.app` 이라는 멋진 무료 도메인이 발급됩니다.

---

## 📱 3. 최종 테스트!
1. 스마트폰이나 외부 컴퓨터 인터넷 주소창에 Vercel에서 발급받은 `https://~~.vercel.app`을 입력합니다.
2. 예쁜 다크모드 "테니스 예약 봇 컨트롤러" 웹사이트가 짠! 나타납니다.
3. 실행 주기 입력란에 `12`를 넣고 [시작하기]를 터치!
4. (잠시 후) 텔레그램으로 *서버에서 보내는 인증 사진과 7자리 번호*가 옵니다! 앱에서 2분 내에 인증을 마칩니다.
5. 이후부터는 내 데스크톱이 꺼져 있어도, 핸드폰이 꺼져 있어도 클라우드 서버에서 12분 간격으로 영구 모니터링하며 빈자리를 스틸해 줍니다. 🎾✨
6. 1스캔 주기(12분)가 끝날 때마다 텔레그램으로 "🤖 [생존 안내] ..." 메시지가 도착하므로 서버가 잘 살아있는지 쉽게 체크할 수 있습니다!
